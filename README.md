  2.4 Introduction
Artificial Intelligence refers to the ability of machines to simulate human intelligence such as learning, reasoning, problem-solving, and decision-making. AI has become an important part of everyday life through virtual assistants, recommendation systems, autonomous vehicles, and smart applications.

Machine Learning (ML) is a subset of AI where systems learn from data instead of being explicitly programmed. Deep Learning is a further subset of ML that uses neural networks with multiple layers to process large amounts of complex data.

Generative AI is a modern advancement in AI where machines generate new content rather than only analyzing existing information. It can create essays, poems, software code, realistic images, synthetic voices, videos, and more. Tools like OpenAI ChatGPT, Google Gemini, and Adobe Firefly are examples of Generative AI systems widely used today.
  2.5 Main Body Sections:

  • Introduction to AI and Machine Learning
Artificial Intelligence aims to create systems capable of performing intelligent tasks. It includes problem-solving, language understanding, vision recognition, planning, and learning.

Machine Learning enables computers to improve their performance using data. Instead of fixed instructions, the model learns patterns from examples.

There are mainly three types of Machine Learning:

Supervised Learning

The model learns using labeled data where both input and output are known. Example: spam email detection.

Unsupervised Learning

The model identifies hidden patterns from unlabeled data. Example: customer segmentation.

Reinforcement Learning

The model learns by trial and error using rewards and penalties. Example: robotics and game-playing AI.

Deep Learning uses artificial neural networks inspired by the human brain. It is highly effective for speech recognition, image generation, and language understanding.
  • What is Generative AI?
Generative AI is a branch of artificial intelligence that creates new content by learning patterns from existing data. Instead of simply identifying or classifying information, it generates outputs that resemble human-created content.

Examples include:

Writing essays and reports
Creating realistic images
Generating music and voice
Producing videos
Writing programming code
Designing 3D models

For example, when a user gives a prompt like “write a poem about nature,” a Generative AI system analyzes learned language patterns and produces a completely new poem.

Generative AI works by training on massive datasets and understanding relationships between words, pixels, sounds, or code structures.
  • Types of Generative AI Models (e.g., GANs, VAEs, Diffusion Models)
A Generative Model is a machine learning model that learns the probability distribution of data and generates new samples similar to the training data.

Types of Generative Models
A Generative Model is a machine learning model that learns the probability distribution of data and generates new samples similar to the training data.

Types of Generative Models
1. Generative Adversarial Networks (GANs)

GANs consist of two neural networks:

Generator
Discriminator

The generator creates fake data, while the discriminator checks whether it is real or fake. Through competition, the generator improves.

Applications:

Face generation
Image enhancement
Deepfake creation
2. Variational Autoencoders (VAEs)

VAEs compress input data into a smaller representation and then reconstruct it. They are useful for generating smooth and meaningful outputs.

Applications:

Image synthesis
Medical imaging
Data compression
3. Diffusion Models

Diffusion models gradually add noise to data and then learn how to reverse the process to generate high-quality outputs.

Applications:

Image generation
Art creation
Video generation

Example: OpenAI DALL·E

4. Transformer Models

Transformers are mainly used in text generation and language tasks. They process words in parallel and understand long-range relationships in text.

Applications:

Chatbots
Translation
Content writing

Example: GPT models

  • Introduction to Large Language Models (LLMs)
A Large Language Model is an advanced AI model trained on huge amounts of text data to understand, predict, and generate human language.

The term “large” refers to:

Large training datasets
Large number of parameters
Large computational power required

Examples include:

GPT
BERT
Gemini
Claude
LLaMA

LLMs can perform:

Question answering
Translation
Essay writing
Coding
Summarization
Conversation
Tutoring

LLMs predict the next most likely word in a sentence using probability and context understanding.

Example:

Input: “The sun rises in the…”

Output: “east”

This prediction becomes highly advanced when scaled to billions of parameters.
  • Architecture of LLMs (e.g., Transformer, GPT, BERT)
Building an LLM involves several important stages.

Step 1: Data Collection

Massive datasets are collected from:

Books
Websites
Research papers
Articles
Code repositories
Documentation

The quality of data directly affects model performance.

Step 2: Data Cleaning

Duplicate, harmful, biased, and low-quality content is removed. Clean data improves reliability.

Step 3: Tokenization

Text is broken into smaller units called tokens. These may be words, subwords, or characters.

Example:

“Artificial Intelligence” → Artificial + Intelligence

Step 4: Model Architecture

Most LLMs use Transformer architecture because it handles long text efficiently.

Step 5: Pre-training

The model learns language patterns by predicting missing or next words using massive computation.

Step 6: Fine-tuning

The model is adjusted for specific tasks such as customer support, healthcare, or education.

Step 7: Reinforcement Learning from Human Feedback (RLHF)

Human reviewers rank responses, helping the model improve quality and safety.
  • Training Process and Data Requirements
The Transformer architecture is the foundation of modern LLMs.

Key Components
Encoder

Processes input and understands meaning.

Decoder

Generates output based on learned context.

Self-Attention Mechanism

Allows the model to focus on important words in a sentence.

For example:

“The animal didn’t cross the road because it was tired.”

The word “it” refers to “animal,” and attention helps identify that relationship.

Positional Encoding

Since transformers process words in parallel, positional encoding helps preserve word order.

Feed Forward Networks

These layers refine understanding and improve prediction quality.

Popular Transformer-Based Models
BERT → bidirectional understanding
GPT → text generation
T5 → text-to-text tasks

GPT is decoder-focused, while BERT is encoder-focused.
